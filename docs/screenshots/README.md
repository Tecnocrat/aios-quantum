# Screenshots Directory

This folder contains timestamped snapshots of AIOS Quantum visualizations.

## Naming Convention

```
aios-quantum_{mode}_{timestamp}.png
```

Examples:
- `aios-quantum_topology_2025-12-13T10-30-00.png`
- `aios-quantum_hypergate_2025-12-13T10-31-00.png`

## How to Capture

1. Open the visualization at `/hypersphere/visualizations`
2. Select your desired mode (Topology Mesh or Hypergate Sphere)
3. Position the camera as desired
4. Click **ðŸ“¸ Snapshot** button
5. Move downloaded file to this folder
